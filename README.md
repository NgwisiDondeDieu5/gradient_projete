
# Algorithme du Gradient Projeté

## Description

Implémentation de la méthode du gradient projeté pour l'optimisation linéaire sous contraintes. L'algorithme permet de maximiser une fonction linéaire avec des contraintes linéaires (égalités et inégalités).

## Fonctionnalités

- Maximisation de fonctions linéaires
- Contraintes d'égalité et d'inégalité
- Interface graphique intuitive
- Visualisation des résultats
- Sauvegarde et copie des résultats

```bash
git clone https://github.com/votre-nom/gradient-projete.git
